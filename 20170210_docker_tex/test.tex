\section{混合分布の平均と標準偏差}

混合分布の平均は以下で示される.

\[\hat{\mu} = \frac{\mu_1 + \mu_2}{2}\]

\[\hat{\sigma}^2 = \frac{\sigma_1^2 + \mu_1^2 + \sigma_2^2 + \mu_2^2}{2} - \hat{\mu}^2\]
